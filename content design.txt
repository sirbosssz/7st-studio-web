Design Guide
พื้นหลังเน้นสีขาว
สีประกอบ: ดำ ส้ม ยีนส์อ่อน
theme: street fashion

แต่ละ Page แบ่ง section ต่างๆ
มีทุก Page
1. Navbar -> พื้นหลังกลืนกับ Hero ในส่วน Home แล้วเปลี่ยนพื้นหลังเป็นสีขาวเมื่อเปลี่ยนหน้าหรือ section
          -> โลโก้อยู่ด้านซ้ายสุด เมนูกับ social อยู่ข้างขวา
2. Footer -> ซ้ายสุดเป็นปีแล้วก็ชื่อสตูดิโอ ขวาสุดเป็นรายชื่อ framework ที่ใช้หรือว่างไว้หรือที่อยู่สั้นๆ
3. Preloading -> อาจจะเป็นไอค่อนอะไรบางอย่างเกี่ยวกับแฟชั่นแล้วม animation อยู่ด้านล่าง

Home
1. Hero -> Studio Logo ตัวใหญ่ๆ
        -> พื้นหลังเป็นภาพ Protrait
        -> แบ่งครึ่งแบบไม่สมดุลกับสีดำ
2. Details  -> เนื้อหาเป็นการอธิบายสตูดิโอคร่าวๆ แบ่งเป็น column
            -> คร่าวๆอาจจะเป็น 3 column มี animation ตอนเลื่อนมาครั้งแรกด้วย หรือเป็น paragraph สั้นๆ
3. User Reviews -> ออกแนวเป็นกล่องๆเหมือน twitter
4. Rewards(ถ้ามี) -> แบ่งภาพข้างนึง ข้อความอีกข้าง แล้วเป็น pagination เลื่อนไปทีละรางวัล

About US
1. Hero -> เป็นรูปกว้างเต็มจอ แต่สูงไม่มาก(อาจจะแนวรูปเกี่ยวกับถ่ายรูป) ตรงกลางเป็นคำว่า About US
2. Details  -> บอกว่าเป็นสตูดิโอถ่ายรูป
            -> รูปข้างนึง ข้อความข้างนึงสลับไปมา

Gallery
1. Gallery(เลือกแบบใดแบบหนึ่ง)
    -> โชว์อัลบัมรูปต่างๆ เป็น grid คล้าย pinterest พอคลิกอัลบัม อัลบัมอื่นๆจะจางลงไป แล้วขยายคอนเทนต์เป็นภาพต่างๆ คลิกแล้วจะแสดงภาพใหญ่
    -> เป็นแนวสไลด์โชว์รูปต่างๆ พอคลิกแล้วแสดงภาพใหญ่

Our Team
1. Team -> แบ่งครึ่ง ข้างนึงบอกเกี่ยวกับทีมงาน อีกข้างเป็นหน้าต่างโชว์รูปแต่ละคน พอเลื่อนเมาส์มาจะเห็นชื่อและหน้าที่

Contact
1. Hero -> เป็นรูปสตูดิโอถ่ายจากข้างนอก สูงไม่มากก็ได้
        -> มุมขวาล่างบอกที่อยู่
2. Social -> โชว์ไอค่อนของ social แต่ละอย่างพร้อมคลิกลิงค์ไปยังเว็บนั้นๆใน tab ใหม่
3. Maps -> โชว์ที่อยู่โดยใช้ google maps api
